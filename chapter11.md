# 构建 Http 服务器

  
本章提要：

* 创建HTTP服务器

*  让HTTP服务器在某个端口上监听

* 处理HTTP 请求

* 观察请求头

*  监听和传送请求数据

*  用状态码和头数据进行响应

* 用主体数据进行响应

* 使用HTTP分块响应流式传送响应主体

HTTP 是一个用来进行传送内容和应用程序的应用层协议， 它将 TCP 用作传输协议，并且是万维网进行数据通信的基础。首选的应用程序部署方案之一就是在互联网上提供 HTTP 服务， 来响应 HTTP 客户端的请求。
